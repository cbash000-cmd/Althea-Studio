ALTHEA Studio - Free Edition

ALTHEA Studio is a mobile-friendly web application designed for real-time music visualization and recording. It lets users create visually stunning videos from their music, featuring multiple overlay modes, customizable colors, and high-resolution export options. This free edition is ideal for experimenting, sharing, or learning how music visualization works on the web.

Features
	•	Audio Input: Supports .mp3, .m4a, and other audio formats.
	•	Static Backgrounds: Add your own images for dynamic visual layers.
	•	Overlay Modes: Six unique visualization styles:
	•	Radial Bars
	•	Standard Bars
	•	Circle
	•	Waveform
	•	Particles
	•	Mirror
	•	Custom Colors & Intensity: Choose primary, secondary, and accent colors, and adjust visualization intensity.
	•	Overlay Images: Add image overlays with adjustable opacity.
	•	Memory Presets: Save and load custom visualizer settings for quick reuse.
	•	Export Video: Record and export your visualizations at:
	•	720p (HD)
	•	1080p (Full HD)
	•	4K (Ultra HD)
	•	Progress Tracker: Visual progress bar during recording.
	•	Fullscreen Mode: Expand the canvas to full screen for better visuals.
	•	Automatic Recording Stop: Recording stops automatically when the song ends.

How to Use
	1.	Open the index.html file in a modern browser.
	2.	Click WAKE SYSTEM to initialize the audio engine.
	3.	Upload an audio file under AUDIO (MP3/M4A).
	4.	Choose a STATIC IMAGE for the background (optional).
	5.	Select your preferred Overlay Mode and adjust colors/intensity.
	6.	Optionally, upload an Overlay Image and set opacity.
	7.	Pick the Export Quality and press PLAY & RECORD.
	8.	Once finished, the video will automatically download.

Notes
	•	This is the Free Edition, featuring a watermark in exported videos.
	•	Designed for mobile-first browsers, but works on desktops.
	•	Future versions may include paid features, advanced effects, and beat detection.
